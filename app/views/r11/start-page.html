Advertise your volunteering opportunities

Use this service to:

  • advertise your opportunities and receive applications through NHS Volunteering
  • advertise your opportunities and link to application forms on existing systems

The process

To advertise an opportunity, you will need to complete the following steps.

1. Enter a title for your listing

This is what potential volunteers will see first when they search for opportunities.

2. Fill in the listing

You will add:

  • a summary
  • the details of the type of person you are looking for
  • an overview of your application process
  • where the opportunity is, or if it's suitable for remote working
  • tags so that users can filter search results
  • the availability you need
  • the contact details where you want to receive queries about the opportunity

3. Select the application settings

You can decide if you want to receive applications through this service or through an external link.

You can set a closing date for this listing.

If you receive applications through this service, you can also set a limit to the number of applications.

Reviewing and making changes

You can save your progress and continue later.

You can also preview the listing before publishing.

Once you publish the listing, you can edit it to change information across the listing.
