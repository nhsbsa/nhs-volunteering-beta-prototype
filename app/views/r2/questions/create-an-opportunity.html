Create an opportunity listing

The information you enter will help promote your opportunities and motivate potential volunteers to register their interest.

To create a listing you will need to complete the following sections.

1. Add a preview

This is what potential volunteers will see first when they search for opportunities.

2. Fill in the listing

You will add:

  • a summary
  • details of the type of person you are looking for
  • an overview of your application process
  • the availability you need
  • the contact details where you want to receive queries about the opportunity

3. Add locations

You can add one or more locations for the opportunity. This will allow potential volunteers can find opportunities that are closest to them.

You will also be able to select if this is a remote opportunity.

4. Select the application settings

You can decide if you want to receive applications through this service or through an external link.

You will set a closing date for this listing.

If you receive applications through this service, you can also set a limit to the number of applications.
